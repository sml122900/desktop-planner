# 2026-05-05 작업 일지

## 완료한 작업

### 초기 프로젝트 구성 (feat: 초기 프로젝트 구성)

Desktop Planner 앱의 전체 기반을 구축했다.

**Main Process**
- `main/index.js`: BrowserWindow (transparent + alwaysOnTop) 생성, Tray 아이콘, setIgnoreMouseEvents로 click-through 구현
- `main/preload.js`: contextBridge로 `window.electron.*` IPC API 노출
- `main/ipcHandlers.js`: schedule / goal / checklist / config 전체 CRUD 핸들러
- `main/scheduler.js`: 60초 폴링 알람 스케줄러, OS 네이티브 Notification 발송

**Renderer**
- `App.jsx`: 레이아웃 3종 분기 (horizontal-split / top-bar-3col / sidebar), 마우스 hover IPC 브리지
- `ScheduleWidget.jsx`: 타임슬롯 목록, 진행 중 이벤트 자동 하이라이트, 완료 토글, 알람 on/off, AddEventForm
- `GoalWidget.jsx`: 목표 프로그레스바, 인라인 값 편집, MiniCalendar, AddGoalForm
- `FocusWidget.jsx`: 현재 포커스, 오늘의 명언 (요일 기반), 체크리스트 CRUD
- `hooks/useSchedule.js`, `useGoals.js`, `useConfig.js`: 상태 관리 + 낙관적 업데이트
- `styles/global.css`: dark-night / warm-paper / frosted 테마 CSS 변수

**데이터 레이어**
- electron-store로 schedules.json / goals.json / config.json 로컬 영속 저장
- 모든 CRUD 핸들러가 갱신된 전체 배열 반환 → 재조회 없이 setState 가능

## 결정 사항

- transparent + alwaysOnTop 창으로 바탕화면 오버레이 구현
- setIgnoreMouseEvents로 위젯 외 영역 click-through 처리
- setInterval 60s 알람 폴링 (node-cron 대신 단순 구조 선택)
- electron-store 로컬 JSON (서버 없이 오프라인 완전 동작)
